## 📖 About

**DSA-A-To-Z** is a comprehensive, community-driven vault of Data Structures and Algorithms problems, solutions, and explanations. Whether you're a complete beginner or preparing for FAANG/product-based company interviews, this repository has everything you need.

### 🎯 Goal
- Cover **every important DSA topic** from scratch
- Provide **clean, well-commented solutions** in multiple languages
- Track problems from **LeetCode, GFG, Codeforces, HackerRank**, and more
- Be a **study companion** for coding interviews and competitive programming

## 🚀 Getting Started

### Prerequisites

Make sure you have any of the following installed:

- **C++** — `g++ --version`
- **Python** — `python3 --version`
- **Java** — `java --version`

### Clone the Repository

```bash
git clone https://github.com/addy12bag/DSA-A-To-Z.git
cd DSA-A-To-Z
```

### Run a Solution

**C++:**
```bash
g++ -o solution solution.cpp
./solution
```

**Python:**
```bash
python3 solution.py
```

**Java:**
```bash
javac Solution.java
java Solution
```

## 🤝 How to Contribute

We ❤️ contributions! Here's how you can help:

### Step-by-Step Guide

1. **Fork** this repository by clicking the Fork button at the top right.

2. **Clone** your forked repo locally:
   ```bash
   git clone https://github.com/YOUR-USERNAME/DSA-A-To-Z.git
   ```

3. **Create a new branch** for your contribution:
   ```bash
   git checkout -b feature/add-problem-name
   ```

4. **Add your solution** under the correct topic folder with this naming convention:
   ```
   ProblemName_PlatformName.cpp   (e.g., TwoSum_LeetCode.cpp)
   ```

5. **Write a comment header** at the top of your solution file:
   ```cpp
   /*
    * Problem: Two Sum
    * Platform: LeetCode (#1)
    * Difficulty: Easy
    * Approach: Hash Map
    * Time Complexity: O(n)
    * Space Complexity: O(n)
    * Author: YourName
    */
   ```

6. **Commit your changes:**
   ```bash
   git add .
   git commit -m "feat: add Two Sum solution (LeetCode #1)"
   ```

7. **Push to your fork:**
   ```bash
   git push origin feature/add-problem-name
   ```

8. **Open a Pull Request** to the `main` branch of this repo.

---

### ✅ Contribution Guidelines

- Follow the folder structure strictly.
- Add **only original or properly attributed** solutions.
- Each solution must include **time and space complexity** in comments.
- Keep code **clean, readable, and well-commented**.
- Avoid duplicate solutions unless using a **different approach**.
- Be respectful in PR discussions — we're all learning here!

### 🐛 Reporting Issues

Found a bug or wrong answer? [Open an issue](https://github.com/addy12bag/DSA-A-To-Z/issues) with:
- Problem name and platform link
- What's wrong with the current solution
- Your suggested fix (optional)
